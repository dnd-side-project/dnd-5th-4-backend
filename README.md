## `commit` 메세지 컨벤션
- 한글을 사용한다.
- 영어 주 행위 키워드는 첫 문자 대문자 (Ex. Add)
- 콜론은 주 행위 키워드와 붙이고, 한글 설명과 한칸 띄운다.
- 예시
  - `Add`: xx기능 추가
  - `Update`: xx기능 수정
  - `Delete`: xx기능 삭제
